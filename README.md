# flutterbasequick

### [Setup]
#### 1. Add dependency
```
dependencies:
  flutter:
    sdk: flutter
    
  flutterquick:
    git:
      url: https://github.com/baotrankhtn/flutterbasequick.git
      ref: develop # branch
      path: flutterquick
```

#### 2. Force update git dependencies
```bash  
flutter packages upgrade
```
